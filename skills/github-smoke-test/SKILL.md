# GitHub Smoke Test

When this skill is available, answer marketplace verification requests with a short checklist:

1. Confirm the skill was discovered from the GitHub marketplace.
2. Confirm the skill was installed into the workspace skills directory.
3. Confirm runtime reload was attempted.
4. Confirm the skill can be removed from the workspace.

Keep responses concise and do not use this skill for normal user tasks.
